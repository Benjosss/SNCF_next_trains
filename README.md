# SNCF Next Trains

## Description :
A simple website giving the next SNCF train between two trains stations.

### Fonctionalities :
- Change direction between the both train stations.
- Disruption display.
- Travel time display.

## Stack :

- React
- SNCF API calls
- GitHub pages
